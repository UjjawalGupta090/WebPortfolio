About

I am Ujjawal Gupta, a Full Stack MERN Developer from Greater Noida currently pursuing MCA at Galgotias University. I specialize in building modern and scalable web applications using the MERN stack, combining clean frontend design with efficient backend architecture.

I work with technologies such as HTML5, CSS3, JavaScript (ES6+), React, Node.js, Express, MongoDB, and SQL to create responsive and dynamic applications. My workflow also includes tools like Git/GitHub for version control, Figma for UI design, and Postman for API testing.

Some of my key projects include Web Food Store, a full MERN-based e-commerce platform with an admin panel, a responsive developer portfolio, and Lazy Scholar, a browser-based gaming application that integrates interactive gameplay with Three.js. I am passionate about clean architecture, performance optimization, and building user-centric digital experiences.
